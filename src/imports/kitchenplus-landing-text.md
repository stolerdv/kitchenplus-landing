### Заголовок

### Подзаголовок
Составляй планы питания для клиентов и делись ими в один клик

### Текст
Создай план питания для клиента в отдельном календаре. Нажми "Поделиться" — приложение сгенерирует красивую мобильную страницу с меню и списком покупок. Отправь ссылку клиенту в мессенджере или соцсетях — никаких PDF, никаких приложений на его стороне.

### Как это работает:
1. Создай дополнительный календарь для клиента
2. Запланируй питание на неделю
3. Нажми "Экспорт" → получи ссылку
4. Отправь клиенту — он видит план и список покупок в браузере

### Блоки преимуществ для специалистов:
- **Экономия времени** — план вместо часа занимает 10 минут
- **Профессиональный вид** — красивая страница с логотипом
- **Вирусный рост** — клиент видит KitchenPlus, скачивает приложение
- **Несколько клиентов** — отдельный календарь для каждого (Pro: 3, Premium: 10)

## FAQ — часто задаваемые вопросы

**Q: Приложение работает без интернета?**
A: Да, базовые функции (просмотр меню, список покупок, рецепты) работают офлайн. Синхронизация происходит при подключении.

**Q: Можно ли использовать без подписки?**
A: Да, бесплатный тариф бессрочный и включает все основные функции для одной семьи.

**Q: Как работает AI-анализ чеков?**
A: Сфотографируй чек в приложении. Наш AI распознаёт купленные продукты и автоматически добавляет их в кладовую. Работает с кассовыми чеками большинства российских магазинов.

**Q: Можно ли поделиться планом питания с кем-то без приложения?**
A: Да. Функция "Экспорт" генерирует ссылку на мобильную веб-страницу — открывается в любом браузере без установки приложения.

**Q: На каком языке работает приложение?**
A: Русский, казахский и английский. Переключение в настройках. Рецепты из каталога также переведены на все три языка.

**Q: Как считается КБЖУ?**
A: На основе базы 500+ ингредиентов с данными пищевой ценности. При добавлении рецепта в меню КБЖУ рассчитывается автоматически с учётом коэффициента порции каждого члена семьи.
